# Optimal Detector 页面优化规格文档

> **Date:** 2026-03-24
> **Project:** optimal-detector UI 优化

## 设计目标

将现有页面从基础 Element Plus 风格优化为**蓝紫渐变毛玻璃拟态**风格。

## 设计方向

- **风格:** 毛玻璃拟态（Glassmorphism）
- **配色:** 蓝紫渐变
- **动画:** 适度动效

## 设计细节

### 1. 整体背景
- 深色蓝紫渐变背景 (#1a1a2e → #16213e → #0f3460)
- 添加动态模糊纹理效果

### 2. 卡片样式
- 半透明毛玻璃背景 (rgba(255,255,255,0.1))
- backdrop-filter: blur(12px)
- 20px 圆角 + 微妙渐变边框
- 悬停时轻微上浮效果

### 3. Header
- 保持深色渐变
- 添加玻璃光泽效果

### 4. 表格组件
- 毛玻璃背景
- 行悬停高亮效果

### 5. 按钮
- 渐变背景
- 发光悬停效果

### 6. 结果区域
- 异常单元发光效果
- 渐变步骤条

## 验收标准

1. 页面整体呈现蓝紫渐变毛玻璃效果
2. 卡片具有半透明模糊背景
3. 按钮有渐变和发光效果
4. 悬停有适度动效
5. 原有功能保持不变
